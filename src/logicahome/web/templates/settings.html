Integrations

Any tool that speaks JSON over HTTP can drive LogicaHome. The full API is at /api/*. Examples:

  • GET /api/devices
  • POST /api/devices/<slug>/state with {"on": true, "brightness": 30}
  • POST /api/scenes/<slug>/run
  • POST /api/scenes/snapshot with {"slug":"bedtime","name":"Bedtime"}

The MCP SSE endpoint for AI clients is at /sse.

Connect an adapter

Each smart-home ecosystem gets its own wizard. The CLI runs the actual flow — this page links you to the right command.

Tuya / SmartLife

Local LAN — needs Tuya IoT Cloud account once.

logicahome connect tuya

Home Assistant

Bridge into an existing HA install via REST.

logicahome connect home-assistant

Philips Hue

Local Bridge REST — press the link button.

logicahome connect hue

Shelly

Manual config (Gen1 + Gen2).

logicahome connect shelly

ESPHome

Native ESPHome API. Optional dep: pip install 'logicahome[esphome]'.

Add devices manually to config.yaml under adapters.esphome.devices.

Matter

Skeleton against python-matter-server. Pair via your existing controller.

Add server_url to config.yaml under adapters.matter.

After connecting any adapter, run:

logicahome discover

then come back to Devices.
